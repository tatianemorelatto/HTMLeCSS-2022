Inserindo Vídeos no site

Essa forma não ocupa tanto trafego de banda no site e fica mais leve.

  1. Acesse o Youtube
  2. Selecione o video e clique em compartilhar
  3. Depois selecione a opção Incorporar
  4. Feito isso, aparcerá o código HTML para colar no site
  5. Você pode definir quando começa o video, o tamanho, largura, etc
  6. Cópia o código e cola no código

Dessa forma não vai consumir dados do seu servidor e sim do Youtube, porém no Youtube o video precisa ser upado lá e qualquer um pode acessar

Temos também o VIMEO porém é pago, então depende do cliente por exemplo
