Estudando imagens!

Basicamente uma imagem é um conjunto de pontos.
A internet não pode ter tamanhos pesados, dependendo do tamanho do arquivo, demora muito para carregar.
O JPEG é um formato com compactação de imagens, tem extensão JPG. (não tem suporte a transparência)
PNG (tem compactação, qualidade melhor que o JPEG- e permite transparência) foi criado pelo W3C, criado parasubstituir o GIF

Imagens em sites

Geralmente uma imagem em um site tem largura de 650px, mas não é uma regra, você deve testar no seu site para validar o tamanho Largura de 1200 px geralmente para usar em fundo
